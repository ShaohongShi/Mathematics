Zermelo-Fraenkel Axioms

All formulations of ZFC are expressed in the symbolism of first order logic, implying that at least one set exists. Suppose $X,Y$ are arbitrary sets,
  1. Extensionality: $\forall x (x \in X \Leftrightarrow x \in Y) \Rightarrow X=Y$
  2. Regularity: $\exists x_1 \in X \Rightarrow \exists x_2 (x_2 \in X \land \neg \exists z (z \in x_2 \land z \in X))$

    No set is an element of itself and that every set has an ordinal rank.

  3. Schema of specification: $\exists \{x \in X : \phi(x)\}$

    4. The empty set now exists and is unique.

  4. Pairing: $\exists Z (X \in Z \land Y \in Z)$

    5. Proved by 6 and 7

  5. Union: $\exists Z(Z=\cup X=\{z: \exists x \in X \land z \in x\})$
  6. Replacement: the image of $X$ under any definable function will also fall inside a set.

    7. Thus we get domain, range and image.

  7. Infinity: there exists a set having infinitely many members.

    8. The minimal set $X$ satisfying the axiom of infinity is the von Neumann ordinal($0=\emptyset,n+1=n \cup \{n\}$).

  8. Power set: For any set $X$, there is a set that contains every subset of $X$, denoted by $\mathcal{P}(x)$

Axiom of Choice

  1. Given a collection $\mathcal{A}$ of disjoint nonempty sets, there exists a set $C$ consisting of exactly one element from each element of $\mathcal{A}$.
  2. Zorn's lemma: A partially ordered set in which every chain has an upper bound has a maximal element.

    3. Suppose there exists a poset $P$ such that every totally ordered subset has an upper bound, and every element has a bigger one. For every totally ordered subset $T$ we may then define a bigger element $b(T)$, because T has an upper bound, and that upper bound has a bigger element.Using $b$, define elements $a_0 < a_1 < a_2 < \cdots$ in $p$. There are too many ordinals (a proper class), more than there are elements in any set, and the set P will be exhausted.

  3. Well-ordered theorem: For any set, there exists an order relation that is a well-ordering.

    4. A well-order on a set $S$ is a total order on $S$ with the property that every non-empty subset of $S$ has a least element in this ordering.

    For any set $X$, by the axiom of choice, there is a function $f$ so that for any nonempty set $Y \subset X, f(Y) \in Y$. By induction on ordinals, define $g$ so that $g(0)=\emptyset$ and $g(\alpha)=f(X- \bigcup_{\beta < \alpha}g(\beta) )$. Then by the replacement and comprehension, the domain of has an upper bound. So $g$ is a function. Now every element of $X$ belongs to the range of $g$. Let $x < y$ if $g^{-1}(x) < g^{-1}(y)$.

Peano Axioms

  1. Axioms
    1. 0 is a natural number.
    2. Equality is reflexive, symmetric, transitive, closed.
    3. For every natural number $n$, $S(n)$ is a natural number.
    4. $S$ is an injection.
    5. There is no natural number whose successor is 0.
  2. Inductive principle Second $(A \subset \mathbb{Z}_+,\forall n \in Z_+, S_n \subset A) \text{ implies } (n \in A) \implies A=\mathbb{Z}_+$.
  3. Well-ordering: $\forall A \in \mathbb{Z}_+,A \ne \emptyset, A$ has a smallest element.
  4. Archimedean ordering: $\mathbb{Z}_+$ has no upper bound in $\mathbb{R}$
