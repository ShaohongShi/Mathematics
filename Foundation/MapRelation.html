Function

  • $f \mid A_0$: restrict $f\colon A \to B$ to $A_0 \subset A$.
  • Composition of $f\colon A \to B, g \colon B \to C$
    • $$g \circ f \colon A \to C, (g \circ f)(a)=g(f(a)) \, .$$
  • Inverse: $f^{-1}, f$ is bijective
  • Injective: $f(a_1)=f(a_2) \implies a_1=a_2$.
  • Surjective: $b \in B \implies \exists a \in A,b=f(a)$.
  • Image: $f(A_0)=\{b \mid \exists a \in A_0,b=f(a)\}, A_0 \subset A$.
  • Preimage: $f^{-1}(B_0)=\{a \mid f(a) \in B_0\},b_0 \subset B$.
  • Support: $\operatorname{supp}(f) = \{i \in X \mid f(i) \ne 0\}$.
  • Multivalued function: every input is associated with at least one output.

Homomorphism Definition

  • A structure-preserving map between mathematical structures.
  • In set theory: functions.
  • In linear algebra: linear transformations.
  • In group theory: group homomorphisms.
  • In topology: continuous functions.

Homomorphism(Category Theory)

  • Monomorphism: $f \colon X \rightarrowtail Y$
    • $$\forall g_1,g_2:X \to Y, f \circ g_1=f \circ g_2 \Rightarrow g_1=g_2 \, .$$
  • Epimorphism: $f \colon X \twoheadrightarrow Y$
    • $$\forall g_1,g_2:X \to Y, g_1 \circ f=g_2 \circ f \Rightarrow g_1=g_2 \, .$$
  • Isomorphism: $f \colon X \to^{\sim} Y$
    • $$\exists g:Y \to X, g \circ f=f \circ g= \mathrm{id}_X \, .$$
  • Endomorphism: a morphism $f \colon X \to X$
  • Automorphism: be both an endomorphism and an isomorphism

Relation

  • Comparability(C): $\forall x \ne y, (xCy)\lor (yCx)$.
  • (Non)reflexivity(R,nR): $\forall x, xCx,(\nexists x, xCx).$
  • Symmetry(S): $xCy \Rightarrow yCx$.
  • Antisymmetry(aS): $xCy \Rightarrow \neg yCx$.
  • Transitivity(T): $(xCy)\land (yCz) \Rightarrow (xCz)$.
  • Negative T(nT): $(\neg xCy) \land (\neg yCz) \Rightarrow \neg xCz)$.

Equivalence

  • Partian: $\mathcal{B}$ $\bigcup\limits_{B \in \mathcal{B}}=A, (B_1 \cap B_2 = \emptyset) \lor (B_1=B_2)$ in $\mathcal{B}$.
  • A set of equivalence classes in $A$ is denoted by $\bar{A}$<./li>
  • (Complete) invariant: $f:A \to B$ is constant (and distinct) on the equivalence classes.
  • Canonical forms: $C \subset A$ has exactly one mumber of each equivalence class.

Order

  • Usual order relation: $x < y$ on $\mathbb{R}$,and $(x,y)=\{a: x < a < y\}$ is an open interval $a < b, (a,b)=\{a < x < b\}$. In addition, if $(x,y)=\emptyset$, $x, y$ are immediate predecessor and immediate successor.
  • Order type: there is a bijective function $f:A \to B, a_1 <_A a_2 \implies f(a_1) <_B f(a_2)$.
  • Dictionary order: $a_1 \times b_1 < a_2 \times b_2$ if $a_1 < a_2$ or if $a_1=a_2, b_1< b_2$.
  • An ordered set has the least upper bound($\sup$) property $\Leftrightarrow$ it has the greatest lower bound($\inf$) property.

Partial Order

  • Extremum: $\forall a \in P$ maximum: $a \le g$, minimum: $a \ge g$.

    • A poset can only have one maximum or minimum.

  • Maximal and minimal: $\nexists a \in P$ maximal: $a > g$, minimal: $a < m$.
  • Bound: $\forall a \in A$, upper : $a \le x$, lower: $a \ge x$.

Order in Sets

  • $\lim\limits_{n \to \infty}{A_n}=\bigcup\limits_{n=1}^{\infty}{A_n}, A_1 \subset A_2 \subset \cdots \subset A_n \subset \cdots$
  • $\lim\limits_{n \to \infty}{A_n}=\bigcap\limits_{n=1}^{\infty}{A_n},A_1 \supset A_2 \supset \cdots \supset A_n \supset \cdots$
  • $\mathop{\overline{\lim}}\limits_{n \to \infty}{A_n}=\bigcap\limits_{n=1}^{\infty} \bigcup\limits_{k=n}^{\infty} {A_k}, \mathop{\underline{\lim}}\limits_{n \to \infty}{A_n}=\bigcup\limits_{n=1}^{\infty} \bigcap\limits_{k=n}^{\infty} {A_k}$
