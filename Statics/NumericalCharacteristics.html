Mathematical Expectation

  • $EX=\displaystyle{\int_{-\infty}^{+\infty}x\mathrm{d}F(x)}$ if $\displaystyle{\int_{-\infty}^{+\infty}|x|\mathrm{d}F(x)}$ exists.
  • Discrete: $EX=\sum\limits_{i=1}^n x_i p_i$ if $\sum\limits_{i=1}^n |x_i| p_i$ exists.
  • Continuous: $EX=\displaystyle{\int_{-\infty}^{+\infty}xf(x)\mathrm{d}x}$ if $\displaystyle{\int_{-infty}^{+\infty}|x|f(x)\mathrm{d}x}$ exists.
  • $E\chi_A=P(A)$.
  • If $Y=g(X)$, which is Borel measurable, $\displaystyle{\int_{-\infty}^{+\infty}y\mathrm{d}F_Y(y)}=\displaystyle{\int_{-infty}^{+\infty}g(x)\mathrm{d}F_X(x)}$.
  • $E(X_1,\cdots,X_n) = (EX_1,\cdots,EX_n), EX_i=\displaystyle{\int_{-\infty}^{+\infty}x_i\mathrm{d}F_i(x_i)}$.
  • If $a \le X \le b$, then $a \le EX \le b$.
  • Linear: $E(aX+b)=aEX+b$.

Variance

  • Variance: $\operatorname{var}(X)=E(X-EX)^2$ if it exists. And $\sqrt{\operatorname{var}(X)}$ is called standard deviation.
  • $\operatorname{var}(X)=EX^2-(EX)^2, \operatorname{var}c=0, \operatorname{var}(aX+b)=a^2\operatorname{var}(X)$.
  • Standardizationï¼š $X^{*}=\dfrac{X-EX}{\operatorname{var}(X)}, EX^*=0, \operatorname{var}(X^*)=1$.
  • If $c \ne EX$, then $\operatorname{var}(X) < E(X-c)^2$.

Correlation coefficient

  • $\sigma_{ij}=\operatorname{cov}(X_i,X_j)=E[(X_i-EX_i)(X_j-EX_j)], \rho_ij=\dfrac{\operatorname{cov}(X_i,X_j)}{\sqrt{\operatorname{var}(X_i)\operatorname{var}(X_j)}}$.
  • $\Sigma=(\sigma_{ij})_{n \times n}$ is a positive-semidefinite symmetric martix.
  • $\rho_{ij} \in [-1,1]$.
  • $\operatorname{cov}(X,Y)=0 \Leftrightarrow EXY=EXEY \Leftrightarrow \operatorname{var}(X+Y)=\operatorname{var}(X)+\operatorname{var}(Y)$.
  • If $X,Y$ are independent, then they are not relative. The inverse is not true.

Inequalities

  • Markov: $P(|X|>\varepsilon) \le \dfrac{E|X|^{\alpha}}{\varepsilon^{\alpha}}, \alpha >0$.
  • Chebyshev: $P(|X-EX|>\varepsilon) \le \dfrac{\operatorname{var}(X)}{\varepsilon^2}, \operatorname{var}(X)=0 \Rightarrow X=c$.
  • Cauchy: If $EX^2,EY^2$ exists, then $E|XY| \le \sqrt{EX^2EY^2}$, equality holds iff $\exists a, b, a^2+b^2 \ne 0, aX+bY \overset{a.s.}{=} 0$.
