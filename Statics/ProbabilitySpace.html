Probability space

  • A measure space $(\Omega,\mathcal{F}, P)$.
  • Sample space $\Omega$: the set of all sample points, which are all possible outcomes or results of of an experiment or random trial.
  • Event field $\mathcal{F}$: a selected $\sigma$-algebra of subsets of $\Omega$, $\omega \in \mathcal{F}$ is a event. $\Omega$ is the certain event, and $\varnothing$ is the impossible event.
  • Probability measure $P$: non-negative, normalized and $\sigma$-additivitive.

    • $\{S_n\} \subset \mathcal{F}$ is a non-decreasing set sequence, then $P(\lim\limits_{n \to \infty}S_n)=\lim\limits_{n \to \infty}P(S_n)$, alongs with finite additive is equaled with $\sigma$-additive.

Conditional probability

  • $P(A|B):={\dfrac {P(AB)}{P(B)}},P(B)>0$.
  • $P(A_1 A_2 \cdots A_n)=P(A_1)P(A_2|A_1) \cdots P(A_n | A_{n-1}\cdots A_1)$.
  • $P(A)=\sum\limits _{n}P(A|B_{n})P(B_{n})$,$\left\{{B_n}\right\}$ partitians $\Omega$.
  • (Bayes)$P(A_{i}\mid B)={\dfrac {P(B\mid A_{i})\,P(A_{i})}{\sum \limits _{j}P(B\mid A_{j})\,P(A_{j})}}, \{A_j\}$ partians $\Omega$.
  • (Continuous)Conditional density function= density function \ margin density function.

Principle of Inclusion and Exclusion

  • $P(\bigcup\limits_{1}^{n}A_i)=\sum\limits_{i}P(A_i)-\sum\limits_{i < j}P(A_i \cap A_j)+\cdots+(-1)^{n-1}P(A_1 \cap \cdots\cap A_n)$.
  • $P(\bigcap\limits_1^nA_i)=\sum\limits_{i}P(A_i)-\sum\limits_{i < j}P(A_i \cup A_j)+\cdots+(-1)^{n-1}P(A_1 \cup \cdots\cup A_n)$.

    • $S_1=\sum\limits_{1 \le i \le n}P(A_i)$, $S_2=\sum\limits_{1 \le i < j \le n}P(A_iA_j)$, $\cdots, S_n=P(A_1 \cdots A_n)$

  • At least $k$ happens: $P=S_k-\dbinom{k}{k-1}S_{k+1}+\dbinom{k+1}{k-1}S_{k+2}- \cdots +(-1)^{n-k}\dbinom{n-1}{k-1}S_n$.
  • Exactly $k$ happens: $P=S_k-\dbinom{k+1}{k}S_{k+1}+\dbinom{k+2}{k}S_{k+2}- \cdots +(-1)^{n-k}\dbinom{n}{k}S_n$.

Random variable

  • $\quad X: \Omega \to \mathbb{R}, \forall x \in \mathbb{R}$, $\{\omega \mid X(\omega) < x\}\in \mathscr{F}$, with cumulative distribution function $F(x)=P\{\omega \mid X(\omega) < x\}, x \in \mathbb{R}$ and value space $A \in \mathcal{B}(\mathbb{R}),P(X \in A)=1$.
  • $F(x)$ is a distribution function $\Leftrightarrow F(x)$ is non-decreasing, left continuous and $F(- \infty)=0,F(+\infty)=1$.
  • Dicrete: There exists countable $x_i \in \mathbb{R}, P(X=x_i)>0, \sum\limits_i P(x_i)=1$.
  • Continuous: $\exists f(x)>0, \displaystyle{F(x)=\int_{-\infty}^x f(t)\mathrm{d}t}, \forall x \in \mathbb{R}$. $f(x)$ is a probability density function $\Leftrightarrow f(x) \ge 0, \displaystyle{\int_{-\infty}^{+\infty}f(x)\mathrm{d}x=1}$

    • $\displaystyle{A\in \mathcal{B}(\mathbb{R}), P(X \in A)=\int_A f(x)\mathrm{d}x=\int_{\infty}^{+\infty}I_A f(x)\mathrm{d}x}$.

    $g(x) \text{ is strictly monotone},g^{-1}(y) \in C^1,f_{Y=g(X)}(y)=|[g^{-1}(y)]^{\prime}|f(g^{-1}(y))$.

  • X \overset{i.d.}{=}Y:=F_X(x)=F_Y(x),\forall x \in \mathbb{R}, X \overset{\text{a.s.}}{=}Y:=P(\{\omega:X(\omega) \ne Y(\omega)\})=0$.

Random vector

Definitions: $X=(X_1, \cdots, X_n), X_i$ are random variables on $(\Omega, \mathcal{F},P)$ with distribution function $F(x_1,\cdots,x_n)=P(\{X_1(\omega) < x_1,\cdots,X_n(\omega) < x_n\})$.
  • $F(x_1,\cdots,x_n)$ is non-decreasing about every variable.
  • $F(x_1,\cdots,-\infty,\cdots,x_n)=0, F(+\infty,\cdots,+\infty)=1$.
  • $F(x_1,\cdots,x_n)$ is left-continuous about each variable.
  • $F(b_1,b_2)-F(b_1,a_2)-F(a_1,b_2)+F(a_1,a_2) \ge 0, \forall a_1 < b_1, a_2 < b_2$.
  • $F(x_1,x_2)$ is a distribution function $\Leftrightarrow F(x_1,x_2)$ satifies $2^0,3^0,4^0$.
  • The uncontinuous points of $F$ can be dense in $\mathbb{R}^n$.

  • $\displaystyle{F(x_1, \cdots, x_n)=\int_{-\infty}^{x_1}\cdots\int_{-\infty}^{x_n}f(t_1,\cdots,t_n)\mathrm{d}t_1 \cdots \mathrm{d}t_n}$.
  • $\forall A \in \mathcal{B}(\mathbb{R}^n), P(X \in A)=\displaystyle{\int_A f(t_1, \cdots, t_n)\mathrm{d}t_1\cdots \mathrm{d}t_n}$.
