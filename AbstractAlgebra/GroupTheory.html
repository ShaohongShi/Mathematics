Homomorphism

  • If $\varphi \in \text{Hom}(G_1,G_2)$, then $\varphi(1)=1,\varphi(a^{-1})=\varphi(a)^{-1}, \operatorname{im} \varphi \le G_2, \ker\varphi \trianglelefteq G_1$
  • If $\varphi : G_1 \twoheadrightarrow G_2$, then $\phi(H)=\varphi(H)$ is a bijective correspondence between $\{H \le G_1: H \supset \ker\varphi\}$ and $\{H \le G_2\}$, and if $H \trianglelefteq G_1, \phi(H) \trianglelefteq G_2$.
  • If $\varphi: G_1 \twoheadrightarrow G_2$, then $G_1/\ker\varphi \cong G_2$.
  • Inner isomorphism: $\sigma_g(a)=gag^{-1}$
  • $\operatorname{Inn}(G)=\operatorname{im}\{\varphi:G \to \operatorname{Aut}(G),g \mapsto \sigma_g\}\trianglelefteq G$.
  • $Z(G)=\ker\{\varphi:G \to \operatorname{Aut}(G),g \mapsto \sigma_g\}\trianglelefteq G$.
